# ifc-live-score-2026

A simple mobile-first live score web app powered by Google Sheets as the backend.

## Data Source

All logic and data are managed in Google Sheets:

https://docs.google.com/spreadsheets/d/1pKrp0JR_f3sVdiWe97V13JwIMOAwJkKWKsP0evkjPvQ/edit?usp=sharing

The spreadsheet acts as the single source of truth for:
- Match schedule
- Live scores
- Standings table
